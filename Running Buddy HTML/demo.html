🏃‍♂️ Running Buddy HTML Project

Project Overview

This is a project that converts a Figma AI-designed React/TypeScript prototype into traditional HTML/CSS/JavaScript. The project uses a modular architecture and includes a complete running partner matching system.

📱 Page Navigation

🔐 Login 🏠 Home (Map & List View) 👤 Runner Profile 🤝 Match Confirmation 📅 Schedule Run

✨ Key Features

🗺️ Map View

Display nearby runners on map with interactive markers

📋 List View

Display runner information in list format with search and filter support

👤 Detailed Profile

Complete runner profile page with statistics and achievements

🤝 Smart Matching

Intelligent matching system based on pace, distance and preferences

📅 Run Scheduling

Flexible time, location and route selection system

📱 Responsive Design

Perfect adaptation for mobile and desktop devices

🛠️ Tech Stack

HTML5 CSS3 JavaScript ES6+ CSS Grid Flexbox CSS Variables Lucide Icons SessionStorage

  1. Click the page links above to start browsing
